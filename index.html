I learned that I know how to think computationally in order to solve problems with code. Becoming aware of that laid a framework for me to confidently approach problems in the future. How to calculate whether a given latlng coordinate is in the bounds of the scope of the map, so that you can reload a the information displayed on the front end when the user moves the map.
